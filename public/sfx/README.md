# Sound Effects Folder

Place your sound effect files (.mp3, .wav, .ogg) in this folder.

The app will randomly pick one to play when a user clicks a song.

Example files you could add:
- bruh.mp3
- yeet.mp3
- oof.mp3
- vine-boom.mp3
- emotional-damage.mp3
- sus.mp3
- fnaf-jumpscare.mp3
- windows-xp-error.mp3
- taco-bell-bong.mp3
- among-us.mp3
